    • O que é Topologia - é um termo para designar a configuração assumida pelos computadores de uma rede ou seja, a forma como estes estão interligados.
    • Topologia física - refere-se à forma como os computadores e cabos de rede se interligam fisicamente na rede. A outra seria a Topologia Lógica, que refere-se ao modo como os dados circulam entre os computadores da rede.


      • Tipos de Topologia

    • Barramento - Computadores interligados por um cabo somente, porém somente um fala com o outro por vez, enquanto dois conversam, o restante fica “barrado” de conversar.
    • Anel - Essa topologia é como um anel mesmo, vários computadores conectados por um cabo somente, um computador está conectado sempre a outros 2. O tráfego de informações é em um sentido somente, por causa disso, se um computador falhar, toda a comunicação pode ser comprometida.
    • Estrela - Nesse tipo, todas as redes são conectadas por um dispositivo central, no caso Switch ou HUB. Por meio desse dispositivo, as estações podem se comunicar entre si. A sua grande vantagem é que se um dos dispositivos falhar, somente ele não irá funcionar, enquanto as outras ainda permanecem conectadas normalmente.
    • Duplo Anel - Mesma coisa do anel, porém como é mais de um cabo, a “conversa” fica ágil, com um cabo recebendo e outro enviando.
    • Estrela Entendida - Igual a estrela, porém um sistema de várias arquiteturas estrela interconectadas.
    • Malha - Dispositivos conectados entre si, cada dispositivo se interliga a todos os outros.
    • Mista - É uma mistura das arquiteturas anteriores interligadas.

  • Transmissão Assíncrona - Envia os dados separadamente com delimitadores de início e fim.
  • Transmissão Síncrona - Enviam os dados em quadros sincronizados com um "relógio" compartilhado entre o receptor e o remetente.

  • Simplex - Forma básica de transmissão, de uma via somente, não há como verificar se todos os dados chegaram ao receptor. Problemas na transmissão não são detectados nem corrigidos. Transmissão em TV aberta é um exemplo disso.
  • Half Duplex - Dados podem ser transmitidos nas duas direções, porém não ao mesmo tempo. A detecção de erros é possível. Comunicação de Walkie-Talkies é um exemplo disso.
  • Full Duplex - A melhor forma de transmissão. Os dados podem ser transmitidos e recebidos simultaneamente. Um bom exemplo é uma conexão de TV a cabo, em que você pode ver TV e navegar na internet ao mesmo tempo.

  • HTTP (Hyper Text Transfer Protocol) - Amplamente utilizado para transmitir informações entre um servidor web e um cliente, como páginas WEB por exemplo. Não possui criptografia.
  • HTTPS (Hyper Text Transfer Protocol Security) - Mesmo que o HTTP, só que é adicionada uma camada de segurança para não haver leitura dos dados por terceiros.
  • FTP (File Transfer Protocol) - Amplamente utilizado para transmitir arquivos entre um servidor e um cliente, mas sem criptografia.
  • FTPS (File Transfer Protocol Security) - Mesmo que o FTP, só que é adicionada uma camada de segurança para não haver interceptação dos dados por terceiros.

    • Cada endereço possui 32 bits, separados em quatro grupos de dois dígitos hexadecimais e pelo símbolo de “.” (ponto). Cada grupo varia entre os números 0 e 255. Possui um limite de 4.294.967.296 possiblidade. Exemplo: 01010101.10101010.01010101.10101010li

    • Cada endereço possui 128 bits, separados em oito grupos pelo símbolo “:” (dois pontos) , cada um contendo quatro dígitos hexadecimais. Cada grupo varia entre o número 0 e 65535. Ele contém muito mais possibilidades de endereço, um número próximo de 3,4 x 10^38 (dez elevado à 38ª potência). Exemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

      Sempre há dois endereços padrões, como no exemplo:

    • 10.0.0.0 - endereço de rede (identificado pelo número 0 no fim)
    • 10.0.0.255 - endereço de broadcast (identificado pelo número 31 no fim)

  • Taxa de Transmissão: Placa Ethernet, que pode ser Ethernet comum, Fast e Giga, que suportam velocidades de transmissão de 10 Mbps, 100 Mbps, 1.000 Mbps, 10.000 Mbps.
  • Cabos de Rede suportados: Ethernet são compatíveis com cabos do tipo par trançado e coaxiais, Token Ring suporta somente par trançado e Fibra ótica.
  • Taxa de Transmissão: Barramento ISA (Industry Standard Architecture),Barramento PC(Peripheral Component Interconnect), e mais outros.
