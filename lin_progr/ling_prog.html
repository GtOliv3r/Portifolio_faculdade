  • IF - Se a condição dentro for verdadeira, executa o bloco de código.
  • ELSE IF - Se a condição IF não for verdadeira, verifica se a condição dentro do ELSE IF é verdadeira, se for, executa o bloco de código interno dele.
  • ELSE - Se a condição IF e a do ELSE IF não for verdadeira, executa o bloco de código do ELSE.
  • SWITCH CASE - Estrutura que usa a sintaxe CASE para se caso algo for true ou false, realiza o bloco de código interno do CASE. Dentro de um SWITCH CASE, pode existir diversos CASES.

(Click)

  • FOR - Para repetição do bloco de código com limite de vezes determinado.
  • WHILE - Para repetição do bloco de código enquanto a condição seja verdadeira até se tornar falso. Pode ser executado no mínimo nenhuma vez.
  • DO WHILE - É executado no mínimo uma vez o bloco interno e se for verdadeira a condição, continua a repetir o bloco de código até se tornar falso.
  • FOREACH - Repete o bloco de código para cada item de um array.
